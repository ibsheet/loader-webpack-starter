# ibsheet-loader-webpack-starter

[`@ibsheet/loader`](https://github.com/ibsheet/loader) starter for webpack

## clone project

```bash
git clone https://github.com/ibsheet/loader-webpack-starter ibl-webpack-starter
cd ibl-webpack-starter
```

## setup and start

#### Using npm:

```
npm install
npm start
```

#### Using yarn:

```
yarn
yarn start
```
